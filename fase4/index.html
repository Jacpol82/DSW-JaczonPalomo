BIENVENIDOS

El curso de Diseño de Sitios Web está dirigido a estudiantes que estén interesados en el diseño web y la administración básica de sitios web,
teniendo en cuenta las herramientas y técnicas de codificación HTML, CSS y frameworks CSS aplicando técnicas de posicionamiento Web.

El curso esta estructurado en 3 unidades didácticas:

  • Unidad 1. Introducción al HTML.
  • Unidad 2. HTML 5 y CSS3.
  • Unidad 3. Frameworks CSS y Posicionamiento web

    • Unidad 3

    Para la unidad tres se desarrollara la Fase 4 Diseño y Construcción III. En la que se hace revisión del diseño y se implementa un framework CSS.

    Objetivo

    Implementar páginas web utilizando un framework CSS aplicando estrategias de posicionamiento para el SEO.

    ¿Qué son las Media Queries?

    Es una técnica que permite condicionar la lectura de distintos estilos, es la que nos abre la posibilidad a un nuevo enfoque de diseño web, que resulta ideal para dar soporte a la enorme diversidad de dispositivos existentes actualmente. Este enfoque es conocido como “diseño web adaptable” o “diseño web sensible” (responsive web design, originalmente).
