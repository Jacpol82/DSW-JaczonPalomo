BIENVENIDOS

El curso de Diseño de Sitios Web está dirigido a estudiantes que estén interesados en el diseño web y la administración básica de sitios web,
teniendo en cuenta las herramientas y técnicas de codificación HTML, CSS y frameworks CSS aplicando técnicas de posicionamiento Web.

El curso esta estructurado en 3 unidades didácticas:

  • Unidad 1. Introducción al HTML.
  • Unidad 2. HTML 5 y CSS3.
  • Unidad 3. Frameworks CSS y Posicionamiento web

    • Unidad 2

    Para la unidad dos se desarrollara la Fase 3 Diseño y Construcción II. En la que se diseña y construye el laoyout del sitio.

    Objetivo

    Desarrollar sitios web utilizando técnicas de codificación HTML y aplicando estilos CSS.

    En esta fase se abordaran los siguientes temas:

    Etiquetas Básicas de HTML

    Estilos con CSS3

    Estructura básica de una página Web (Layout)

    Los cuales seran aplicados para el desarrollo de la actividad.
